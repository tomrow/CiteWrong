*CiteWrong*

A basic and terrible bibliography manager for Windows platforms. 

CiteWrong is built using version 3.5 of the .NET Framework and is compatible with Windows XP and newer systems.

This program uses XML to load and save bibliographies and export schemes. Sample documents demonstrating how the program structures these, as well as a demonstration of the use of export scheme macros, are provided in this repository.

To compile, load the project in Visual Studio and use the standard Compile function. No external dependencies are needed
