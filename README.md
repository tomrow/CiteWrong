*CiteWrong*

A basic and terrible bibliography manager for Windows platforms. CiteWrong is built using version 3.5 of the .NET Framework and is compatible with Windows XP and newer systems.
CiteWrong uses XML-based formats to load and save bibliographies and export schemes. Sample documents are provided in this repository.

To compile, load the project in Visual Studio and use the standard Compile function. No external dependencies are needed
